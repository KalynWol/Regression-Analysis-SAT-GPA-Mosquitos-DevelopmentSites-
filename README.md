# Regression-Analysis-SAT-GPA-Mosquitos-DevelopmentSites-
A hypothesis test for effect of SAT score on GPA and mosquito pupae and development sites. 
The first part evaluates the relationship between SAT scaores and GPA. The second part evaluates the relationship between the number of breeding sites and the total population of mosquitoes.  
Both conclusions show there are strong relationships between the varaiables being tested and end up rejecting their null hypotheses. 
The document and Rstudio code provide a comprehensive analysis of the assignment.
